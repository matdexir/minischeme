# Minischeme

It is a scheme-like language written in python as a way to learn more about programming language design with no to little outside dependecy

## How to use

Just run the following and you should get a scheme REPL

```bash=
python main.py
```

## Basic usage

```
minischeme> (* 10 30)
minischeme> (* 10 (+ 5 25))
```
